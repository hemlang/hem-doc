# Форматирование кода

Hemlock включает встроенный форматировщик кода для обеспечения единообразного стиля.

## Использование

```bash
hemlock format <FILE>         # Форматировать файл на месте
hemlock format --check <FILE> # Проверить, отформатирован ли файл (код выхода 1, если нет)
```

## Правила стиля

Форматировщик применяет следующие соглашения:

| Правило | Значение |
|---------|----------|
| Отступы | Табуляция |
| Стиль фигурных скобок | K&R (открывающая скобка на той же строке) |
| Максимальная ширина строки | 100 символов |
| Завершающие запятые | Да, в многострочных контекстах |
| Максимальное количество последовательных пустых строк | 1 |

## Автоматический перенос строк

Форматировщик автоматически переносит длинные строки:

- **Параметры функций** - Длинные списки параметров разбиваются по одному параметру на строку
- **Бинарные выражения** - Длинные цепочки логических/сравнительных операций разбиваются по операторам
- **Операторы import** - Длинные списки импортов разбиваются с каждым элементом на отдельной строке
- **Цепочки методов** - Длинные цепочки разбиваются перед точками

## Пример

До:
```hemlock
fn create_user(name: string, email: string, age: i32, active: bool, role: string) { return { name: name, email: email, age: age, active: active, role: role }; }
```

После:
```hemlock
fn create_user(
	name: string,
	email: string,
	age: i32,
	active: bool,
	role: string,
) {
	return {
		name: name,
		email: email,
		age: age,
		active: active,
		role: role,
	};
}
```

## Интеграция с CI

Используйте `--check` в CI-конвейерах для обеспечения форматирования:

```bash
hemlock format --check src/main.hml || echo "File not formatted"
```
