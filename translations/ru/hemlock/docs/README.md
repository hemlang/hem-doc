## Краткий справочник

### Hello World
```hemlock
print("Hello, World!");
```

### Базовые типы
```hemlock
let x: i32 = 42;           // 32-битное знаковое целое
let y: u8 = 255;           // 8-битное беззнаковое целое
let pi: f64 = 3.14159;     // 64-битное число с плавающей точкой
let name: string = "Alice"; // Строка UTF-8
let flag: bool = true;     // Булево значение
let ch: rune = '🚀';       // Кодовая точка Unicode
```

### Управление памятью
```hemlock
// Безопасный буфер (рекомендуется)
let buf = buffer(64);
buf[0] = 65;
free(buf);

// Сырой указатель (для экспертов)
let ptr = alloc(64);
memset(ptr, 0, 64);
free(ptr);
```

### Асинхронность/Параллелизм
```hemlock
async fn compute(n: i32): i32 {
    return n * n;
}

let task = spawn(compute, 42);
let result = join(task);  // 1764
```

## Философия

Hemlock всегда придерживается принципа **явное вместо неявного**:
- Точки с запятой обязательны
- Ручное управление памятью (без сборщика мусора)
- Необязательные аннотации типов с проверкой во время выполнения
- Небезопасные операции разрешены (это ваша ответственность)

Мы даём вам инструменты для безопасной работы (`buffer`, аннотации типов, проверка границ), но не заставляем их использовать (`ptr`, ручное управление памятью, небезопасные операции).
